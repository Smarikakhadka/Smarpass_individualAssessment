# 🔐 SmartPass

##  What the Project Does

SmartPass is a web-based password strength checker that analyzes user passwords in real-time.

It evaluates password strength based on:
- Length
- Use of uppercase and lowercase letters
- Numbers
- Special characters
- Common password detection

The system provides:
- Strength classification (Weak, Medium, Strong)
- Feedback on issues in the password
- Suggestions to improve security
- Estimated crack time
- Machine learning-based prediction of password strength
